# Color converter to Android. 
Change the style of color to Android style or Web style (0xFF) from iOS style(0..1).
iOSで使用されている 0〜1までの色指定をAndroidやWebで使われている１６進数の形式に変換します

ex:)

iOS: R:0.2, G:0.1, B:0.3, A:0.4

Android: #331a4d 

Web: #1a4d33
